# 🎲 Peste Écarlate

Bienvenue sur le dépôt officiel de **Peste Écarlate**, un jeu de rôle axé sur la survie viscérale et l'adaptation dans un monde dévasté par la montée des eaux et une mystérieuse brume corrosive.

## 📜 Présentation
Dans un futur où la fonte des glaces a englouti les civilisations, l'humanité survit péniblement face à la **Peste Écarlate** — une menace organique qui dévore tout. Les survivants se sont divisés en trois philosophies :
* **Les Suprématistes** : Transcendent la biologie par la cybernétique et les drones.
* **Les Harmonistes** : Ont fusionné avec la Peste pour survivre dans les zones infestées.
* **Les Puristes** : Maintiennent l'héritage technologique et humain de l'Ancien Monde par la discipline.

Le système de jeu privilégie la fluidité et l'immersion : pas de calcul complexe, une résolution basée sur l'équilibre entre votre **Inné** (Attributs) et votre **Acquis** (Vocations), où chaque jet de dés est une ressource précieuse.
